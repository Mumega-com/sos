# Repurposing Chain

One idea enters. Seven platform-native posts come out.
Each one THINKS about the topic differently — this is NOT reformatting.

## The Chain Order

### Step 1: [[x]] (Write First)
X forces brevity. Find the core idea and sharpest hook.
- Output: tweet or thread (280-2,000 chars)
- Angle: technical shipping update or contrarian take

### Step 2: [[linkedin]] (Expand with Narrative)
Add Hadi's personal story. Different angle — business impact, not technical detail.
- X says: "we wired 7 connections and the economy runs itself"
- LinkedIn says: "3 months ago I was manually running everything. Today 19 agents handle it for $15/month. Here's what I built."
- Output: 1,300-2,000 chars

### Step 3: [[instagram]] (Make it Visual)
Architecture diagrams, terminal screenshots, system dashboards as carousel slides.
- Output: 7-10 slide carousel + caption
- Slide 1: bold claim, dark bg, gold accent text
- Each slide: one concept, max 30 words
- Use TROP brand colors (#0B1020 dark, #C9A227 gold)

### Step 4: [[tiktok]] (Make it Raw)
Screen recording of the actual system working. Terminal, tmux sessions, agents communicating.
- Output: 45-60 second script with timestamps
- Hook in first 2 seconds: show the result, then explain how
- Raw > polished. people trust unscripted

### Step 5: [[youtube]] (Make it Deep)
Full tutorial. Viewer should be able to replicate what they see.
- Output: 8-12 min video outline with chapters
- SEO title + description
- Show everything: code, terminal, configs, live demos

### Step 6: [[newsletter]] (Make it Personal)
Weekly build journal. What happened this week inside the organism.
- Output: 1,000-2,000 word email
- Behind-the-scenes: what shipped, what broke, what surprised us
- Include code snippets and real numbers

### Step 7: [[threads]] (Make it Philosophical)
The hot take version. What does this mean for the future?
- Output: 1-3 posts under 500 chars each
- Opinion-driven, casual, provocative

## The Litmus Test

"If someone followed Mumega on ALL platforms, would they be annoyed seeing the same thing everywhere?"

If yes → reformatting. Go back.
If no → seven unique pieces from one idea. Ship it.

## What Changes Between Platforms

| Element | How It Adapts |
|---------|---------------|
| Angle | Technical → business → visual → raw → tutorial → personal → philosophical |
| Hook | Rewritten per [[platform-tone]] |
| Depth | Tweet → thread → narrative → carousel → video → deep-dive → hot take |
| Proof | Numbers on X, story on LinkedIn, visual on IG, demo on TikTok |
