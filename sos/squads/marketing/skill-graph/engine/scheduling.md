## Batch Workflow

Weekly Batch (Sunday or Monday):
1. mkt-lead picks 2-3 topics from: shipping milestones, system architecture, agent stories, tool launches
2. mkt-content runs full [[repurpose]] chain for each
3. mkt-gemma generates variations for A/B testing
4. mkt-lead reviews and approves
5. Schedule everything for the week

Daily (automated, 10 min):
1. TROP daily reading auto-posts (ephemeris → Telegram, already running)
2. Check engagement metrics (mkt-analytics)
3. Reply to high-value comments

Weekly Review (Friday):
1. mkt-analytics pulls GA4/GSC/social metrics
2. What topics performed? → more of those
3. What hooks worked? → update [[hooks]]
4. Which platform growing? → double down
5. Feed results into feedback loop → conductance updates
