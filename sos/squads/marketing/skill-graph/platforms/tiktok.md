# TikTok

## Platform DNA
- Video: 45-90 seconds sweet spot
- Vibe: raw, unpolished, authentic. overproduced = worse performance
- Audience: [[curious]] primarily. discovering AI, not deep-diving yet

## Content Rules
- Hook in FIRST 2 SECONDS. show the result, then explain
- Screen recordings of terminal: agents talking, tasks dispatching, code generating
- Text overlays mandatory (most watch muted)
- No long intros. jump straight in
- Match [[brand-voice]] at most energetic. See [[platform-tone]]

## Formats
1. Terminal Recording — show agents working in tmux. voiceover explains
2. Quick Demo — "watch this AI build a page in 30 seconds"
3. "I Replaced X With Y" — "$8k team → $15/month of agents"
4. System Tour — walk through the architecture in 60 seconds

## Script Template
[HOOK: 2 sec — "19 AI agents run my business while I sleep"]
[CONTEXT: 5 sec — "here's what that actually looks like"]
[DEMO: 30-40 sec — terminal recording, agents communicating]
[RESULT: 5 sec — "all automated. $15 a month."]
[CTA: 3 sec — "follow for more agent economy content"]

## Posting: 5-7x/week, daily. Peak: 12:00-15:00, 19:00-22:00 UTC
