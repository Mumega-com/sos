# Instagram

## Platform DNA
- Caption: 2,200 chars, but carousels drive engagement
- Vibe: visual-first. dark backgrounds, gold accents, architecture diagrams
- Audience: [[builders]] + [[curious]] — wider, more visual learners

## Content Rules
- Carousels: 7-10 slides, one idea per slide
- Slide 1 = hook. bold claim, large text, max 8 words
- Brand colors: #0B1020 (void black), #C9A227 (gold), white text
- Caption adds context slides don't cover
- Hashtags: 5-10 at end. #AIagents #agenticAI #opensource #buildinpublic
- CTA last slide: "Follow @mumega for more" / "Save this"

## Formats
1. Architecture Carousel — system diagram broken into slides
2. Before/After — manual vs automated, old vs new
3. Agent Roster — "Meet the team" with agent cards
4. Reels — cross-posted from [[tiktok]] terminal recordings

## Posting: 3-4x/week. Peak: 16:00-18:00, 23:00 UTC
