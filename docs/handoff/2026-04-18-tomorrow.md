## The subagent squad — what works

Confirmed: **custom subagent specs in `.claude/agents/*.md` are NOT dispatchable via the Agent tool.** Only globally-registered subagents are. Workaround that worked:
- Use `subagent_type: general-purpose` with `model: sonnet` override
- Write detailed task-specific prompts inline
- Parallelize 8+ dispatches per tool-call message
- Sonnet at temperature 0.1 produced good JSON Schema + Pydantic + tests

**Architectural gate review** via `subagent_type: Athena` worked perfectly — Athena caught 4 real drift issues in Sprint 1 that I would have missed. Use her at every major commit gate.

Cost economics: Sonnet subagent for schema authoring ~$0.15 per call. Sprint 1 total: ~$4.50. Sprint 2 total: ~$2. Well inside $164/5hr budget.
