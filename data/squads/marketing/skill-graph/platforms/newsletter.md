# Newsletter

## Platform DNA
- Email via Inkwell/Resend. we own the audience
- 800-1,500 words. weekly build journal format
- Vibe: personal letter from inside the organism. most intimate platform

## Content Rules
- Subject line = hook. use [[hooks]] for email. "We Wired a Complete Economy This Week" not "Newsletter #12"
- Open with what happened this week. raw, specific, real
- Structure: 3 things we shipped + 1 thing that broke + 1 thing we learned
- Include code snippets, terminal output, real numbers
- End with ONE CTA: reply, check the repo, try a tool
- Plain text or minimal design. no fancy templates

## Posting: 1x/week, Thursday 13:00-15:00 UTC
