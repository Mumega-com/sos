# Audience: Curious

## Who They Are
Tech-adjacent professionals discovering the agentic economy. Marketers, PMs, founders who've used ChatGPT but haven't built with AI agents. Following the trend, not yet building.

## What They Want
- "Wow" moments: "wait, 19 AI agents run this business?"
- Simple explanations: what agents are, how they work, why it matters
- Entry points: "here's where to start if you want to try this"
- Aspirational but achievable: "you could build this over a weekend"

## How to Talk to Them
- Simpler language. explain MCP, bus, tmux on first mention
- More encouraging, less technical depth
- Visual demos over code snippets
- "Imagine if..." energy followed by "here's how it works"
